Single Column Layout

This layout uses the default alignment of stacked articles which are used as containers to hold the images and their related textual content. Within each article (container) is an image wrapped with a figure tag that is floated. Every other image is floated to the opposite (left or right) side of the container using a CSS rule:


        .float-right {
            flex-flow: row-reverse;
        }

Since I am using FLexbox to control the layout, it is fairly easy to reverse the order of the items within the box.

Transform: skewY(-6deg)

The code that allows for the color background to be shown on a slant is a CSS3 property called transform. There are several values for this property. The one used here is skewY and is applied to a background element separate from the content.

Transform Code

The code for the transformation is shown below. The .diagonal-box class assigns the element to a relative position. Then we apply a rule ::before that creates the skewed diagonal-box with a background color.


                        .diagonal-box {
                            position: relative;
                        }
            
                        .diagonal-box::before {
                            content: "";
                            position: absolute;
                            height: 18em;
                            top: 0;
                            right: 0;
                            left: 0;
                            bottom: 0;
                            background-color: #B2D732;
                            transform: skewY(-6deg);
                        }
